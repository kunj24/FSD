This repository contains **7 practical projects** organized in separate folders, each demonstrating different aspects of full-stack development:

## 🚀 Quick Start

### Prerequisites
Make sure you have the following installed:
- [Node.js](https://nodejs.org/) (v14 or higher)
- [npm](https://www.npmjs.com/) (comes with Node.js)
- [Git](https://git-scm.com/)

### Clone the Repository
```bash
git clone https://github.com/kunj24/FSD.git
cd FSD/Practical
```

## 📂 Project Details

### 🌐 **Pr1-2** — HTML Fundamentals
- **Type**: Static HTML pages
- **Purpose**: Basic HTML structure and CSS styling
- **Run**: Open HTML files directly in browser

### ⚛️ **pr3** — React with Vite
- **Type**: React + Vite
- **Purpose**: Modern React development with fast build tooling
- **Features**: Component-based architecture, hot reload
- **Run**: `cd pr3 && npm install && npm run dev`

### ⚛️ **pr4** — React Application
- **Type**: React
- **Purpose**: React components, state management, and routing
- **Features**: Interactive UI components
- **Run**: `cd pr4 && npm install && npm start`

### 🖥️ **pr9** — Express.js Basics
- **Type**: Express.js + Node.js
- **Purpose**: Basic server setup with routing
- **Features**: RESTful API structure, middleware
- **Run**: `cd pr9 && npm install && npm start`

### 📊 **pr10** — Express.js with Logging
- **Type**: Express.js + EJS
- **Purpose**: Advanced server with error logging and log viewer
- **Features**: Error handling, log file management, admin panel
- **Run**: `cd pr10 && npm install && npm start`

### 👥 **Pr11** — Team Onboarding Template
- **Type**: Express.js + EJS
- **Purpose**: Professional team dashboard with API endpoints
- **Features**: `/home` route, team information API, onboarding workflow
- **Run**: `cd Pr11 && npm install && npm start`

### 🧮 **pr12** — Kids Calculator
- **Type**: Express.js + EJS + CSS Grid
- **Purpose**: Interactive calculator with responsive design
- **Features**: 
  - ✨ Single-page layout (no scrolling)
  - 📱 Mobile-responsive design
  - 🎨 Professional UI with CSS Grid
  - ➕➖✖️➗ Basic arithmetic operations
  - 🚀 Real-time calculations
- **Run**: `cd pr12 && npm install && npm start`

## ⚡ Running Projects

### For Express.js Projects (pr9, pr10, Pr11, pr12)
```bash
cd [project-folder]
npm install
npm start
# Open http://localhost:3000
```

### For React Projects (pr3, pr4)
```bash
cd [project-folder]
npm install
npm run dev  # for Vite (pr3)
# or
npm start    # for standard React (pr4)
# Open the URL shown in terminal
```

### For HTML Projects (Pr1-2)
```bash
cd Pr1-2
# Open .html files directly in your browser
```
